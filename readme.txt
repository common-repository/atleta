== Description ==

The Atleta plugin makes it possible to easily embed registration forms on your website. Works perfectly on mobile as well as on desktop.

= About Atleta =
Atleta is a service with which you easily manage registrations for your sports event. It is suitable for all types of
sports: cycling, running, walking, swimming, and more. Create a ticket for every (race) category, distance or discipline
of your event. Want to register participants in teams? No problem. Sell extras such as T-shirts? Just add them. Besides
this, you get powerful tools such as time slots, coupons, down payments, group tickets, chargebacks, and a lot more.

== Installation ==

This section describes how to install the integration.

If you are a first-time user of Atleta:

1. Go to [Atleta](https://atleta.cc) and create an account.
2. Create your first project.
3. Create an event for which you want to sell tickets and products.
4. Publish your event by clicking on the event, and pressing the "Live" toggle.
5. Get the link of your checkout by clicking on three dots, and selecting "Vist page". It should look like `https://atleta.cc/e/{EVENT_ID}`.

To active the integration on your website.

1. Go to your WordPress admin page.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Create an event in Atleta, press the three dots, and select "Visit page"). Get the link of your checkout, it should look like `https://atleta.cc/e/{EVENT_ID}`.
4. In any of your posts, reference this checkout page. Clicking on any link referencing the checkout page will activate the integration.

That's it!

== Frequently Asked Questions ==

= How do I use this plugin? =

Simply enable the plugin, and if any link in your website references the Atleta checkout, the embed will take care and appear before your user.
